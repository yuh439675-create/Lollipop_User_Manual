短剧APP使用教程（用户端）

本教程面向： 普通用户（观看短剧、充值会员、邀请好友的观众） 系统名称： ReelChain 短剧平台 适用平台： iOS、Android APP

📖 目录

  1. 什么是短剧APP
  2. 如何注册和登录
  3. 如何观看短剧
  4. 如何充值金豆

1. 什么是短剧APP

短剧APP 是一个专门用来观看短视频剧集的手机应用。就像看电视剧一样，但每一集只有几分钟，非常适合碎片时间观看。

主要功能：

  • 🎬 看短剧：海量短剧资源，随时随地观看
  • 🪙 金豆充值：用金豆解锁单集、整部短剧、购买会员
  • ❤️ 收藏关注：喜欢的短剧可以收藏，方便下次观看

2. 如何注册和登录

2.1 下载APP

首先需要下载短剧APP到手机上：

  • 苹果手机：在 App Store 搜索APP名称下载
  • 安卓手机：通过提供的下载链接安装

2.2 注册账号

APP支持多种注册方式，选择一种最方便的即可：

方式一：邮箱注册

  1. 进入我的

  2. 点击设置

  3. 选择"邮箱注册"

  4. 点击Inscreva-se agora

  5. 输入你的邮箱地址

  6. 输入验证码

  7. 输入新密码并确认密码

  8. 输入邀请码（已有邀请码就不用填）

  9. 点击 Inscreva-se agora 注册完成

  10. 点击Account password去登录

  11. 输入账号和设置的密码和Entrar进行登录

  12. 登录完后进入首页

方式二：第三方账号登录（最快捷）支持以下方式快速登录：

  • 谷歌账号登录（Google）

    1. 进入我的

    2. 点击设置

    3. 选择Sign in with Google进行谷歌第三方登录

    4. 登录完后进入首页

💡 小提示：使用第三方账号登录后，系统会要求你绑定手机号，这样更安全。

3. 如何观看短剧

3.1 浏览短剧

打开APP后，你会看到4个主要tab（底部导航栏）：

🏠 首页

  • 推荐短剧：系统会把精彩的视频放到推荐

  • 横幅广告位：轮播展示热门短剧

  • 分类筛选：按类型找短剧（悬疑、言情等）

  • 查询短剧和热门排行榜：最火的短剧

📺 推荐

  • 视频流模式，像刷抖音一样刷短剧
  • 上下滑动切换不同短剧片段

⭐ 追剧

  • 你正在追的短剧都在这里（收藏的短剧）
  • 方便快速找到上次看到哪一集

👤 我的

  • 个人信息

  • 会员状态

  • 钱包余额：查看剩余金豆

  • 余额明细：查看增加或减少金豆的原因

  • 观看历史：快速找到上次看的剧

  • 邀请好友：点击保存二维码发给朋友

3.2 观看短剧步骤

第一步：选择短剧

  1. 在首页找到喜欢的短剧
  2. 点击短剧封面图
  3. 进入短剧详情页

第二步：查看详情

在详情页可以看到：

  • 短剧简介：剧情介绍

  • 集数列表：全部集数，比如第1-100集

  • 收藏/分享按钮

第三步：选择集数观看

  • 点击某一集，比如"第1集"
  • 进入播放页面开始观看

3.3 免费观看 vs 付费观看

免费观看：

  • 每部短剧前几集是免费的（通常前1-10集）
  • 可以先看免费集数，觉得好看再决定是否付费

付费观看：

后面的集数需要付费，有两种方式：

  1. 购买会员：会员可以免费看所有短剧（推荐！）

  2. 用金豆解锁：单独购买某一集

  3. 用金豆解锁：一整部剧

4. 如何充值金豆

如果不想买会员，可以充值金豆，用金豆解锁单集短剧。

4.1 充值金豆步骤

第一步：进入充值页面

  1. 点击"我的"tab
  2. 点击"钱包"或"金豆余额"
  3. 点击"充值"按钮

第二步：选择充值套餐

系统提供多档金豆套餐：

  • 💰 10金豆 = $9.9
  • 💰 22金豆 = $19.9
  • 💰 53金豆 = $49.9
  • 💰 255金豆 = $199.9

💡 小提示：金额越大，赠送的金豆越多

第三步：使用金豆购买

  • 选择支付方式（同会员购买）
  • 完成支付
  • 金豆立即到账

4.2 使用金豆观看

  1. 观看需要付费的集数
  2. 系统提示"需要XX金豆解锁"
  3. 点击"金豆解锁"
  4. 金豆自动扣除，集数解锁成功

7. 常见问题解答

Q1: 注册时收不到验证码怎么办？

答：

  • 检查手机号是否输入正确
  • 查看短信是否被拦截（看"垃圾短信"文件夹）
  • 等待1-2分钟，验证码可能延迟
  • 尝试用邮箱注册
  • 联系客服解决

Q2: 购买会员后还需要金豆吗？

答：

  • 不需要！会员可以免费看所有短剧
  • 金豆和会员选一个即可
  • 建议：经常看短剧的用户买会员更划算

Q3: 短剧看到一半，下次如何继续看？

答：

  • 系统会自动记录观看进度
  • 下次打开该短剧，会从上次位置继续播放

Q4: 如何收藏喜欢的短剧？

答：

  1. 进入短剧详情页

  2. 点击"收藏"按钮（⭐图标）

  3. 收藏的短剧在"我的→收藏"中查看

Q5: 短剧看不了 / 播放失败怎么办？

答：

  • 检查网络连接（建议用Wi-Fi观看）
  • 切换清晰度（点击播放器设置）
  • 重启APP再试
  • 更新APP到最新版本
  • 联系客服反馈

Q6: 邀请的好友没有绑定我的邀请码？

答：

  • 好友需要在注册时填写邀请码
  • 如果注册时忘记填写：
    1. 进入"我的→设置"
    2. 找到"绑定邀请人"
    3. 输入你的邀请码
    4. 提交绑定

Q7: 如何联系客服？

答：

  1. 进入"我的"页面

  2. 点击"帮助中心"或"在线客服"

  3. 可以查看常见问题（最后一个图圈住的信息可点击）

Q8: 可以退款吗？

答：

  • 虚拟商品（会员、金豆）一般不支持退款

Q9: 账号安全如何保障？

答：

  • 设置复杂密码（不要用生日、123456等）
  • 不要把账号密码告诉别人
  • 绑定手机号和邮箱，方便找回密码
  • 定期修改密码

Q10: 如何注销账号？

答：

  1. 进入"我的→设置"

  2. 找到"账号注销"

  3. 阅读注销须知

  4. 确认注销（注意：注销后数据无法恢复）
